# LTX Studio -- Cloud-Based AI Video Production Platform

## Overview
LTX Studio is Lightricks' flagship cloud-based creative development platform for AI video production. It provides a full production suite used by creators and media teams to design, iterate, and produce video projects using AI.

## URL
- **Website:** https://ltx.studio
- **Playground:** https://app.ltx.studio/ltx-2-playground/t2v

## Key Features
- Text-to-video generation
- Image-to-video generation
- Audio-to-video with voiceovers and character dialogue
- In-app video editing module
- XML export to Premiere Pro, DaVinci Resolve
- AI voiceover and lip sync
- Character consistency across scenes
- Camera movement control
- Keyframe-based animation

## LTX-2 Integration
LTX-2 is now available on the LTX platform, powering the cloud-based production tools.

## Distinction from Open-Source
LTX Studio is the commercial platform built on LTX Video models. It is distinct from:
- **LTX-Video (LTXV)** -- open-source model weights
- **LTX-2 / LTX-2.3** -- open-source audio-video models
- **LTX Desktop** -- open-source desktop application
- **ComfyUI-LTXVideo** -- open-source ComfyUI nodes
