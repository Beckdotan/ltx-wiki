# LTX-Video Spatial Upscaler 0.9.7

**Source:** https://huggingface.co/Lightricks/ltxv-spatial-upscaler-0.9.7

---

## Overview

The LTX Video Spatial Upscaler is a diffusion-based model designed to enhance the spatial resolution of videos generated by LTX Video models. It specifically upscales latent representations of videos produced by LTX Video generation models.

## Architecture

- **Model Type:** Latent Diffusion Video Spatial Upscaler
- **Developer:** Lightricks
- **Input:** Latent video frames from an LTX Video model
- **Output:** Higher-resolution latent video frames
- **Upscaling Factor:** 2x on both height and width dimensions

## Compatibility

Works with:
- `Lightricks/LTX-Video-0.9.7-dev`
- `Lightricks/LTX-Video-0.9.7-distilled`
- Likely also compatible with 0.9.8 variants

## Constraints

- **Resolution:** Works best under 720x1280
- **Resolution divisibility:** Must be divisible by 32
- **Frame count:** Divisible by 8 + 1 (e.g., 257 frames max)
- **Auto-padding:** Non-compliant inputs padded with -1, then cropped

## Recommended Workflow

The spatial upscaler works best in a 3-step pipeline:

### Step 1: Generate at Lower Resolution
```python
latents = pipe(
    conditions=[condition1],
    prompt=prompt,
    negative_prompt=negative_prompt,
    width=downscaled_width,
    height=downscaled_height,
    num_frames=num_frames,
    num_inference_steps=30,
    output_type="latent",
).frames
```

### Step 2: Upscale Latents
```python
upscaled_latents = pipe_upsample(
    latents=latents,
    output_type="latent"
).frames
```

### Step 3: Denoise Upscaled Video (Recommended)
```python
video = pipe(
    conditions=[condition1],
    prompt=prompt,
    negative_prompt=negative_prompt,
    width=upscaled_width,
    height=upscaled_height,
    num_frames=num_frames,
    denoise_strength=0.4,
    num_inference_steps=10,
    latents=upscaled_latents,
    output_type="pil",
).frames[0]
```

### Step 4: Downscale to Expected Resolution
```python
video = [frame.resize((expected_width, expected_height)) for frame in video]
export_to_video(video, "output.mp4", fps=24)
```

## Full Code Example

```python
import torch
from diffusers import LTXConditionPipeline, LTXLatentUpsamplePipeline

# Load base model and upsampler
pipe = LTXConditionPipeline.from_pretrained(
    "Lightricks/LTX-Video-0.9.7-distilled", 
    torch_dtype=torch.bfloat16
)
pipe_upsample = LTXLatentUpsamplePipeline.from_pretrained(
    "Lightricks/ltxv-spatial-upscaler-0.9.7",
    vae=pipe.vae,  # Share VAE between models
    torch_dtype=torch.bfloat16
)
pipe.to("cuda")
pipe_upsample.to("cuda")
```

## Key Design Decisions

- Operates in latent space (not pixel space) for efficiency
- Shares VAE with the base generation model
- Optional denoising pass after upscaling improves quality significantly
- The denoise_strength=0.4 preserves structure while refining details
