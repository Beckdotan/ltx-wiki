# ID-LoRA: Identity-Driven Audio-Video Personalization with In-Context LoRA

## Metadata
- **Title:** ID-LoRA: Identity-Driven Audio-Video Personalization with In-Context LoRA
- **Authors:** Amit Dahan, Maya Yanuka, Niv Kraicer, Lior Wolf, Raja Giryes
- **Affiliation:** Tel Aviv University; Lightricks (collaboration)
- **Date:** March 2026 (arXiv: 2603.10256)
- **Source URL:** https://arxiv.org/abs/2603.10256

## Key Details (from AVControl references)

ID-LoRA extends the AVControl framework to identity-driven audio-video personalization. It uses the parallel canvas LoRA approach to enable identity preservation in generated audiovisual content.

## Significance to LTX Ecosystem

- Extends the AVControl framework to identity-driven personalization
- Demonstrates that the LTX-2/AVControl ecosystem can be adapted for person-specific generation
- Addresses one of the noted capability gaps of AVControl (identity preservation)
- Uses the In-Context LoRA (IC-LoRA) paradigm from the LTX-2 model family
- Collaboration between Tel Aviv University and Lightricks

## Notes

This paper was identified through the AVControl paper (reference [16]) as a concurrent work adopting the AVControl framework. It addresses identity-driven audio-video personalization -- generating videos that preserve a specific person's appearance and voice characteristics.
