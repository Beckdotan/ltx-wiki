## Overview
- LTX Desktop is an open-source desktop app for generating videos with LTX models
- Runs locally on supported Windows/Linux NVIDIA GPUs
- API mode available for unsupported hardware and macOS
- Released alongside LTX-2.3 (March 5, 2026)
- Licensed under Apache 2.0

## Key Features Praised by Community

### Generation + Editing in One Interface
- "Generation and editing live in the same interface -- you prompt directly on the timeline"
- Text-to-video works up to 720p natively (1080p with optional upsampler)
- Synchronized audio and video generation in a single pass
- Audio-conditioned generation: feed it a track, visuals move to it

### NLE Integration
- XML timeline import from DaVinci Resolve, Premiere Pro, and Final Cut Pro
- "Changes how editors can use it as a mid-production tool"
- Positions LTX Desktop as a complement to existing editing workflows, not a replacement

### Zero Cloud Cost
- No per-generation cost when running locally
- No internet required after initial setup
- Only paid component: API-based generation (for macOS or unsupported hardware)

## Community Assessment

### Positive Reception
- "The fastest path to usable local AI video generation as of March 2026"
- "The first local AI video editor that actually made reviewers stop and think: okay, this is different"
- "A 22-billion-parameter open-source video and audio generation model that rivals closed commercial tools -- at zero cloud cost"

### Hardware Concerns
- Official requirement: 32 GB VRAM (high barrier for many users)
- Community workaround: RTX 3070 laptop (8GB VRAM) can run with community-optimized fork
- Reduced resolution settings and slower generation on lower-end hardware
- 720p ceiling on budget hardware

### LTX Desktop vs ComfyUI Discussion
- Active community debate about which is better
- Desktop offers simplicity and integrated timeline
- ComfyUI offers more flexibility and customization
- Consensus: both have a place; Desktop for quick results, ComfyUI for advanced workflows

## Sentiment
