## Platform Presence

### Cloud Deployment Platforms

LTX Video has been adopted by multiple third-party cloud platforms:

- **Replicate** -- Text-to-video and image-to-video
- **fal.ai** -- API access
- **Modal** -- Cloud infrastructure with CLI, API, and web UI
- **WaveSpeed** -- Optimized deployment
- **ThinkDiffusion** -- Cloud ComfyUI with LTX support

### Community Ecosystem

- Active LoRA ecosystem on Civitai and HuggingFace
- Official LTX-Video-Trainer repository maintained by Lightricks for community fine-tuning
- Kijai's community-maintained quantized variants widely used
- NVIDIA featured LTX in their official video generation guide
