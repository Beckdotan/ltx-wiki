# RunComfy

RunComfy provides managed, browser-based ComfyUI hosting with pre-loaded LTX Video workflows. It offers a more user-friendly interface than raw GPU rental services like [[runpod]], with both a traditional Playground UI and a full ComfyUI workspace.

**Website:** https://www.runcomfy.com
**LTX workflows:** https://www.runcomfy.com/comfyui-workflows/ltx-2-comfyui-workflow-real-time-video-generation-speed

See also: [[inference-providers-overview]], [[runpod]]

## Available LTX Video Workflows

- **LTX-2 ComfyUI Workflow** -- Real-time video generation speed
- **LTX-2.3 Image-to-Video** -- Realistic motion workflow
- **LTX-2 Video Retake** -- Modify video with AI for precise retakes

## Features

- Pre-loaded LTX Video workflows (no manual model setup)
- Playground interface for easy model access
- Full ComfyUI workspace for advanced users
- No local hardware required
- Persistent storage options
- LTX 2 API access with unified parameters
- Browser-based LoRA training

## Pricing

- Pay-as-you-go plan for getting started
- Pro plan with advanced features (persistent storage, CPU machines)

## When to Choose RunComfy

RunComfy is best suited for:
- Users who want ComfyUI without installation or GPU setup
- Quick experiments with LTX Video workflows
- Non-technical users who prefer a visual interface
- Artists and designers working with node-based workflows

For raw GPU access and custom deployments, see [[runpod]]. For programmatic API access, see [[fal-ai]] or [[replicate]].
